<product_requirements>
The application is a Facebook-like social sharing and chat platform for Uber and Lyft drivers, initially focused on communication. Core features requested were user authentication, a social feed, messaging, and user profiles. The application was to be in English.

Subsequent feature requests included:
1.  **Referral System:** Unique codes for inviting others, tracking referrals.
2.  **Friends System:** Send/accept/decline requests, optional messages, no friend limit, badges for requests/unread messages.
3.  **Advanced Post Sharing:** Public, Friends-only, or Specific friends privacy options.
4.  **Like/Dislike with Auto-Delete:** Thumbs up/down for posts/comments; auto-delete if 10+ votes and 50%+ are dislikes. Users cannot vote on own posts.
5.  **Group System:** Create groups, invite/join, approval-based/open, admin roles, group-specific posts.
6.  **Forgot Password:** Password reset functionality.
7.  **Sign Out Confirmation Removal:** Direct sign-out without confirmation.
8.  **Profile Picture:** Add/edit profile pictures.
9.  **Help Section:** FAQ-style page detailing app features.
10. **Settings Page:** Manage password, delete account, language selection.
11. **App Name Change:** Driver Forum to Drivers Chat.
12. **Data Reset:** Ability to clear all user and post data.
13. **Terms & Conditions:** Two mandatory agreements before login/register, covering non-commercial use, data, and content policies.
14. **Admin Post Deletion:** Admin users can delete inappropriate posts.
15. **About Page & Persistent Terms:** Dedicated About page and a viewable Terms & Conditions page.
16. **Post Edit & Image Enhancements:** Users can edit their posts, and images should be viewable full-screen (not cropped).
